<Note>
  This endpoint accepts a **project API key** (`AS_` prefix). The webhook must belong to the authenticated project.
</Note>

<RequestExample>

```bash cURL
curl --request DELETE \
  --url https://api.autosend.com/v1/webhooks/60d5ec49f1b2c72d9c8b1234 \
  --header 'Authorization: Bearer AS_your-api-key'
```

---
#### Authorizations
<ParamField path="Authorizations" type="string | header" required>
  Project API key header of the form Bearer `AS_<key>`.
</ParamField>

### Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the webhook to delete.

  Example: `"60d5ec49f1b2c72d9c8b1234"`
</ParamField>

#### Response

<span className='text-sm'>Webhook deleted successfully (200)</span>

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message

  Example: `"Webhook deleted successfully"`
</ResponseField>

#### Error Responses

<ResponseField name="404 - Webhook not found" type="object">
  Returned when no webhook with the given ID exists in the project.

  ```json
  {
    "success": false,
    "error": {
      "message": "Webhook not found",
      "code": "WEBHOOK_NOT_FOUND",
    }
  }
  ```
</ResponseField>
