Once youâ€™ve created a transactional email template, you can use it in your application via the AutoSend API.

### API Endpoint

```jsx
POST https://api.autosend.com/v1/mails/send
```

### Authentication

Include your API key in the Authorization header:

```jsx
Authorization: Bearer YOUR_API_KEY
```

### Single Email with Template

**Request:**

```jsx
curl -X POST https://api.autosend.com/v1/mails/send \  -H "Authorization: Bearer YOUR_API_KEY" \  -H "Content-Type: application/json" \  -d '{    "to": {      "email": "customer@example.com",      "name": "John Doe"    },    "from": {      "email": "hello@mail.yourdomain.com",      "name": "Your Company"    },    "templateId": "template_abc123xyz",    "dynamicData": {      "first_name": "John",      "company_name": "AutoSend",      "verification_link": "https://yourapp.com/verify?token=xyz"    }  }'
```

**Response:**

```jsx
{  "success": true,  "message": "Email queued successfully",  "data": {    "emailId": "email_xyz789",    "status": "queued"  }}
```
